# История изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## 1.0.0 — 2026-04-18

### Добавлено

- Первая публичная версия Auto Clicker Vue.
- Авто-клик с настраиваемым интервалом (10–60000 мс).
- Поддержка пяти типов действий: ЛКМ, ПКМ, СКМ, прокрутка колеса вниз, нажатие Enter.
- Глобальные горячие клавиши для старт/пауза: `F8`, средняя кнопка мыши, ЛКМ, ПКМ.
- Авто-пауза при движении мыши с настраиваемой длительностью «остывания» (1–60 с).
- Сохранение всех настроек в `localStorage` через `useLocalStorage` из VueUse.
- Реактивный счётчик кликов в текущей сессии.
- Тёмная тема Nuxt UI 4 «из коробки».

### Технологии

- Tauri 2 + Rust на бекенде; низкоуровневые WinAPI-хуки (`SetWindowsHookEx`) для глобальных хоткеев.
- Vue 3.5, Pinia 3, VueUse 12, es-toolkit на фронтенде.
- Nuxt UI 4 (standalone Vue plugin) поверх Tailwind CSS 4 и Reka UI.
- Vite 8, TypeScript 6, Vitest 4.
- GitHub Actions: тесты, сборка, релизы под Windows / Linux / macOS.
